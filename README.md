# tabela-de-urgencias

Tabela usada para organização de nomes de paicentes.

Essa tabela é um projeto em desenvolvimento criado pela necessidade de armazenar o nome do paciente, bem como outros dados, para que seja dada a devida prioridade em determinado exame. Foi criado para ser utilizada na área da saúde, mas também pode ser adptada para outro tipo de serviço.
